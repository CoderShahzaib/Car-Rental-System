# Car Rental System WebAPI

A RESTful WebAPI built using **.NET Core 10** for managing car rentals, bookings, and customer data. This project supports CRUD operations for Cars, Customers, and Bookings, and is ready to integrate with a frontend application.

---

## Features

- Manage Cars (CRUD)
- Manage Customers (CRUD)
- Manage Bookings (CRUD)
- RESTful API endpoints
- JWT-based authentication support
- Environment-based configuration for security
- Easily deployable to any hosting platform

---

## Tech Stack

- **Backend:** .NET Core 10, C#
- **Database:** SQL Server
- **Authentication:** JWT
- **Version Control:** Git & GitHub
- **Frontend Integration:** Can be used with Angular, React, or any other frontend
